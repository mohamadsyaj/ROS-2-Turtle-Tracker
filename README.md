# Turtle Tracker

Turtle Tracker is a ROS 2 Python package for `turtlesim`. The package runs a turtle-chasing node that tracks a target turtle, drives the main turtle toward it, and respawns the target after it is reached.

## Project Summary

This project demonstrates ROS 2 robotics concepts using Python and `turtlesim`, including:

- Creating and running a ROS 2 Python package
- Building a package with `colcon`
- Using a ROS 2 console script entry point
- Controlling turtle movement in `turtlesim`
- Tracking a target turtle and respawning it after capture

## Requirements

- ROS 2 installed and sourced
- Python 3
- `turtlesim`
- `colcon`
- `setuptools`

Install `turtlesim` if needed:

```bash
sudo apt install ros-$ROS_DISTRO-turtlesim
```

## How to Build

From the root of your ROS 2 workspace:

```bash
colcon build --packages-select turtle_tracker
```

After building, source the workspace:

```bash
source install/setup.bash
```

## How to Run

Start `turtlesim` in one terminal:

```bash
ros2 run turtlesim turtlesim_node
```

In another terminal, source your workspace and run the tracker:

```bash
source install/setup.bash
ros2 run turtle_tracker turtle_chaser
```

## How It Works

The project is designed to run a ROS 2 node that controls a turtle inside the `turtlesim` simulator. The main turtle follows a target turtle. When the target is reached, the target is respawned so the chase can continue.
